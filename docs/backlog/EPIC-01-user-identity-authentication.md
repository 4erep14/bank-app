## EPIC-01: User Identity & Authentication

**Goal:** Enable customers to securely register, authenticate with 2FA-SMS, manage their credentials, and maintain their profile — forming the security foundation for all other epics.

**Scope:**
- Included: Customer self-registration, email/password login, 2FA via SMS (stubbed), password reset, profile view/update, account lockout
- Excluded: OAuth2/SSO, biometric login, KYC document upload, admin-created accounts

**Stories:** US-001, US-002, US-003, US-004, US-005
