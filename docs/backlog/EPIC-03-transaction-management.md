## EPIC-03: Transaction Management

**Goal:** Enable customers to transfer funds between their own accounts and view complete transaction history, while giving admins system-wide transaction visibility.

**Scope:**
- Included: Internal fund transfers (own accounts only), transaction history with filtering, transaction details, admin transaction overview
- Excluded: External/SWIFT/SEPA transfers, bill payments, scheduled/recurring transfers, transaction disputes, CSV/PDF export

**Stories:** US-010, US-011, US-012, US-013
